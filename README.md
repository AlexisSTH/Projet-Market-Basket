# Market Basket Analysis — Instacart

## Objectif business
Identifier les produits fréquemment achetés ensemble
pour proposer des recommandations cross-sell
et optimiser le placement produit en rayon.

## Dataset
Instacart Market Basket Analysis (Kaggle)
~3M de commandes, 50k+ produits

## Stack technique
- Python (pandas, mlxtend, matplotlib, seaborn)
- Algorithme Apriori
- Jupyter Notebook

## Étapes
1. Cadrage business
2. Exploration & nettoyage
3. Transformation en matrice panier
4. Algorithme Apriori
5. Interprétation & recommandations
